Mi Proceso de Aprendizaje en Desarrollo Web

Reflexiones metacognitivas semana a semana

Semana 1: Fundamentos de las tecnologías web

¿Qué aprendí?

Comprendí los conceptos básicos de cómo funciona la web, incluyendo la diferencia entre páginas, sitios, aplicaciones y sistemas web. Aprendí sobre los protocolos fundamentales como HTTP y TCP/IP, y los roles en el desarrollo web.

¿Cómo lo aprendí?

Mediante la creación de diagramas conceptuales, ejemplos prácticos de comunicación cliente-servidor, y la implementación de pequeñas páginas HTML para ver los conceptos en acción.

Semana 2: HTML5, CSS3 y Diseño Responsivo

¿Qué aprendí?

Profundicé en la estructura semántica de HTML5 y las propiedades avanzadas de CSS3 como Flexbox y Grid. Aprendí a crear diseños responsivos que se adaptan a diferentes dispositivos.

¿Cómo lo aprendí?

Practicando con ejercicios de maquetación, utilizando las herramientas de desarrollador del navegador para probar diferentes enfoques, y creando proyectos pequeños con requisitos específicos de diseño.

Semana 3: Bootstrap y Tailwind CSS

¿Qué aprendí?

Aprendí a utilizar frameworks CSS para agilizar el desarrollo. Bootstrap con su sistema de grid y componentes preconstruidos, y Tailwind con su enfoque utility-first.

¿Cómo lo aprendí?

Recreando diseños existentes usando ambos frameworks, comparando sus ventajas y desventajas, y personalizando temas para entender su configuración.

Semana 4: JavaScript, TypeScript y Canvas

¿Qué aprendí?

Adquirí habilidades en programación con JavaScript, incluyendo el manejo del DOM y Canvas para gráficos. También aprendí los beneficios de TypeScript para proyectos más grandes.

¿Cómo lo aprendí?

Implementando pequeños juegos con Canvas, transformando código JavaScript a TypeScript, y resolviendo problemas algorítmicos para practicar los conceptos.

Semana 5: Frameworks JavaScript

¿Qué aprendí?

Comprendí la arquitectura de frameworks frontend como React, Vue o Angular, y diferentes enfoques para manejar estilos en aplicaciones complejas.

¿Cómo lo aprendí?

Construyendo la misma aplicación pequeña con diferentes frameworks, experimentando con CSS Modules, Styled Components y otras técnicas de estilado.

Semana 6: Eventos y Consumo de APIs

¿Qué aprendí?

Aprendí a manejar eventos complejos, flujos condicionales y a consumir APIs REST usando diferentes métodos (fetch, axios).

¿Cómo lo aprendí?

Creando aplicaciones que interactúan con APIs públicas, implementando diferentes patrones de manejo de estado, y practicando el manejo de errores.

Semana 7: Hooks en React

¿Qué aprendí?

Dominé el uso de hooks como useState, useEffect y useReducer para manejar estado y efectos secundarios en componentes funcionales.

¿Cómo lo aprendí?

Refactorizando componentes de clase a funcionales con hooks, creando hooks personalizados para lógica reutilizable, y resolviendo problemas comunes con ejemplos prácticos.

Reflexión General del Curso

¿Qué aprendí en general?

A lo largo de estas semanas he adquirido un conocimiento completo del desarrollo web moderno, desde los fundamentos hasta técnicas avanzadas. Ahora entiendo cómo todas las piezas (HTML, CSS, JavaScript, frameworks) trabajan juntas para crear aplicaciones web completas.

¿Cómo fue mi proceso de aprendizaje?

Mi aprendizaje fue progresivo y acumulativo. Comencé con conceptos básicos que luego se volvieron fundamentales para entender temas más complejos. La combinación de teoría con práctica constante, la construcción de proyectos progresivamente más desafiantes, y la reflexión sobre mis errores fueron clave para mi comprensión.

¿Qué haré diferente en el futuro?

Documentaré mi aprendizaje desde el inicio, crearé más proyectos personales para aplicar los conceptos, y participaré más en comunidades de desarrollo para aprender de otros.
